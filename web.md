# html

## 简述

html是超文本标记语言

## 基本结构

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Hello world</title>
  </head>
  <body>
  </body>
</html>
```



## 常见标签

- 标题<h1></h1>
- <a href="http://baidu.com" target="_blank"></a>
- <p></p>
- <img src="" width=""  height="">
- <table><tr><th></th></tr></table>
- 列表<ol></ol>  <ul><ul>
- <div></div>  <span> </span>
- 表单<form action="" method="post">
- 下拉列表<select></select>
- 属性：id（独一无二的） , class（表示一类）
